Installation - production environment

If Solity Platform is already installed Follow these steps

1. Before you Start

If the Solity Platform is already installed, please make sure that the app is not running.

1. Closing Solity

Double Tap the home button and drag the Solity iOS Platform to the top of the screen. If you can't find the Solity iOS Platform, it may not be running.

2. Environment Selected

Ensure that the "Training" environment is selected.

Previous Next

2. Install the Platform

Click the link bellow to install the Solity Platform on iOS.

1. Running for the first time

If you are installing the app for the first time, and running iOS 9 or above, then you need to allow the Solity Platform to run on your device.

2. Configuring Profile

Open the iOS "Settings" app and select General->Profile.

3. Selecting the Profile

Click on the entry for "Solity Software Ltd".

4. Giving Trust Permissions

Click the "Trust Solity Software Ltd" button.

5. Confirm Permission

In the pop-up window, click "Trust".

6. iOS ready

You can now open the Solity Platform from the home screen.

3. After installation

Click the link bellow to install the Solity Platform on iOS

1. General Options on the Settings device

Select Settings from the bottom options device and select General option then select Solity Profile

2. Trust on Solity

Select the blue Trust button on the top of the app.

3. Confirm

A new pop-up window will appear asking trsut on Solity, select Trust.
